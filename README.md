# 🦙 LLaMA (Large Language Model)

LLaMA (Large Language Model Meta AI) é uma família de modelos de linguagem de grande escala criada pela Meta (Facebook).  
Ela é usada para tarefas de inteligência artificial como geração de texto, tradução, chatbots e análise de linguagem natural.  
Os modelos LLaMA são conhecidos por serem eficientes, de código aberto e capazes de rodar localmente em computadores e servidores.
